# IAM (iam)

Identity and Access Management (IAM) is a framework of policies and technologies that ensures the right users have appropriate access to technology resources. IAM encompasses authentication, authorization, and the management of digital identities across systems. This index tracks IAM-related APIs, standards, and developer resources.
